# event-tags
event-tags uses information from last.fm to augment events on songkick genres of these events as tags and statistical popularity scores using unique listeners.

## Idea
Event listings are overwhelming and having more information that is useful to determine taste and popularity helps user to user the event listing effectively to find interesting ones.

## Features
1. Adds number of unique listeners of artists
2. Adds genre of the music produced by artists
3. Adds visual cues for popularity as guitar symbols to mark artists with 200k, 500k and 800k listeners
